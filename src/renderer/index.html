      出力ハイライト / キーワード通知
      
      
      
        
      
        
      
          ・1行＝1キーワード（部分一致）。既定は黄色ハイライト
      
          ・=色 で色指定：yellow / red / green / cyan / magenta / blue（例 Down=red）
      
          ・行頭に ! を付けると一致時に通知（ビープ音＋トースト＋タブ点滅）
      
          ・# で始まる行はコメント。全セッション共通で適用されます
        
      
      
      
      
      
    
      
  
      

  
  
      
    
      
      
      マクロ
      
      
      
        
      
          
      
            
            
          
      
        
      
        
      
        
      
          wait "文字列" … 受信待ち（既定10秒）｜ timeout 秒 … 以降のwait上限
      
          send "文字列" … そのまま送信｜ sendln "文字列" … 末尾に改行を付けて送信
      
          pause 秒 … 待機｜ print "メモ" … 端末に表示｜ # … コメント
      
          エスケープ: \n \r \t \\ \" \xHH が使用可
        
      
        
      
          Tera Term の .ttl を実行（実用サブセット）。対応: 変数/式・if/elseif/else/endif・while・for/next・do/loop・
      
          goto/:label・call/return・break/continue｜ sendln/send/wait/waitln/wait4all/pause/mpause/flushrecv
      
          inputbox/passwordbox/getpassword/yesnobox/messagebox/statusbox｜ str2int/int2str/strlen/strcompare/strconcat/copy/strscan ほか
      
          ※ connect/ファイル操作などホスト外の命令は未対応（無視）。timeout 変数で wait のタイムアウト秒を指定。
        
      
      
      
      
      
    
      
  
      

  
  
      
    
      
      
      ファイル転送（XMODEM / YMODEM）
      
      
      
        
      
          
        
      
        
      
          
        
      
        
      
          開始後、相手側（ルータ/サーバ）で対応コマンドを実行してください。
      
          XMODEM受信は保存ファイル名を指定、YMODEM受信は保存先フォルダを選びます。
